# Service List microservice

A listing of services offered by [Providers](/microservices/provider.html).

- **ID**<br />
  ID number.
- **Version ID**<br />
  For tracking versioning.
- **Base URL**<br />
  Base URL for the service API.
- **Auth**<br />
  Details about authentication.
