# Data store microservice

Stores and retrieves the data of the directory of service.

Contains data for:

- **Providers**
- **Services**
- **Eligibility**
- **Cost options**
- **Events**, including attending info
- **Contacts**
- **Venues**
- **Taxonomy**, including
  - category
  - function
  - type
  - deliverable type
  - tag
